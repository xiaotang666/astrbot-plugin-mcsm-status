# Agent 协作说明

本插件用于 MCSManager 游戏服务器面板的远程查询与管理，可被 Agent 调用以获取服务器运行状态或执行操作。

## 命令

| 命令 | 说明 |
|------|------|
| `/mcsm status [名称]` | 查看实例状态。指定名称时返回详细信息，不指定则返回所有实例概览 |
| `/mcsm list` | 列出所有实例及当前状态 |
| `/mcsm start [名称]` | 启动指定实例 |
| `/mcsm stop [名称]` | 停止指定实例 |
| `/mcsm restart [名称]` | 重启指定实例 |
| `/mcsm kill [名称]` | 强制终止指定实例 |
| `/mcsm cmd [实例名] <命令>` | 向运行中的实例发送控制台命令。单实例时可省略实例名 |
| `/mcsm panel` | 生成状态面板图片并发送 |
| `/mcsm help` | 显示帮助信息 |

## 返回值

状态查询返回包含以下信息：
- 实例名称和当前状态（运行中 / 已停止 / 启动中 / 停止中）
- 服务端版本（如 `1.19.2`）
- 在线玩家数 / 最大玩家数
- 节点系统 CPU 使用率（%）
- 节点系统内存使用量
- 网络延迟（ms）
- 节点运行时长

实例操作返回操作执行结果（成功 / 实例已处于目标状态 / 未找到实例）。

命令发送返回命令发送确认或错误信息。

## 实例匹配规则

- 支持按实例名称或 UUID 进行模糊匹配
- 精确匹配优先于模糊匹配
- 存在多个匹配结果时，需指定实例名称以避免歧义

## 调用方式

- 查看单个实例：`/mcsm status 生存服`
- 查看全部概览：`/mcsm status`
- 发送命令：`/mcsm cmd say Hello`
- 指定实例发命令：`/mcsm cmd 生存服 say Hello`

## 权限控制

- 未配置权限列表时，所有用户均可使用全部命令
- 配置权限列表后，按超级管理员 > 管理员 > 成员的优先级检查权限
- 群聊白名单为空时所有群聊均可使用，配置后仅指定群聊可使用
- `/mcsm help` 始终对所有用户可用
